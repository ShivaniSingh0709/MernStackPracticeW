# 🚀 Projects Included

## 🎮 Simon Game

A browser-based memory game built using **HTML, CSS, and JavaScript**.

Features:

* Color pattern recognition
* Increasing difficulty levels
* Interactive UI

---

## 🎵 Spotify Clone

A simple **Spotify UI clone** built using HTML and CSS.

Features:

* Sidebar navigation
* Music player layout
* Basic responsive design

---

## 🧠 JavaScript Practice

This folder contains **practice files for core JavaScript concepts** such as:

* Promises
* Fetch API
* DOM manipulation
* Async JavaScript

Files included:

* `app.js`
* `index.html`
* `styles.css`

---

## ⚙️ Express Basics

A basic Express.js project used to learn:

* Creating an Express server
* Handling routes
* Sending responses
* Basic backend setup

---

## 📄 EJS Templating

Practice project to understand **server-side rendering with EJS**.

Concepts covered:

* EJS template engine
* Rendering dynamic pages
* Passing data from server to views

---

## 📦 Node Packages (Figlet)

Practice project to learn how to **install and use Node.js packages from npm**.

Example package used:

* **Figlet** – generates ASCII text in the terminal.

---

## 🔄 REST API CRUD Operations

A simple CRUD application built with **Node.js, Express.js, and EJS**.

Features:

* Create a post
* Read all posts
* Update existing posts
* Delete posts
* Dynamic page rendering with EJS

### Run the Project

Install dependencies:

```
npm install
```

Start the server:

```
node index.js
```

Open in browser:

```
http://localhost:3000
```
